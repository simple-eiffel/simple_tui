Overview

simple_tui provides a modern terminal-based UI framework for Eiffel applications. Features EiffelVision2-compatible API (extend/prune naming), 256-color support, mouse input, keyboard navigation, and Unicode rendering.

EV Compatible

Familiar API for EiffelVision2 developers

256 Colors

Full terminal color palette support

Mouse & Keyboard

Complete input handling with focus management

Unicode

Box drawing and special characters

Quick Start

class MY_APP

inherit
    TUI_APPLICATION
        redefine
            setup
        end

create
    make

feature {NONE}

    setup
        local
            box: TUI_BOX
            btn: TUI_BUTTON
        do
            create box.make (40, 10, "My App")
            create btn.make_with_text ("Click Me")
            btn.set_on_click (agent on_click)
            box.extend (btn)  -- EV-compatible API
            root.extend (box)
        end

    on_click
        do
            quit
        end

end

Layout Containers

TUI_BOX

Container with optional border and title. Can nest other widgets.

create box.make_with_title ("Settings", 40, 20)
box.set_padding (1)
box.extend (label)
box.extend (field)

TUI_VBOX / TUI_HBOX

Vertical and horizontal flow layouts.

create vbox.make (60, 20)
vbox.set_gap (1)  -- Spacing between children
vbox.extend (row1)
vbox.extend (row2)

Styling

Colors

local
    style: TUI_STYLE
    color: TUI_COLOR
do
    create style.make_default

    -- Named colors
    style.set_fg (color.cyan)
    style.set_bg (color.blue)

    -- 256-color palette
    create color.make_index (202)  -- Orange
    style.set_fg (color)

    -- Attributes
    style.set_bold (True)
    style.set_reverse (True)
    style.set_underline (True)
end

Applying Styles

button.set_normal_style (my_style)
button.set_focused_style (focused_style)
button.set_pressed_style (pressed_style)
